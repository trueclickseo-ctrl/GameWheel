Pillar 4: Reference Glossary

Wheel Game Encyclopedia

The definitive dictionary of decision-wheel concepts, probability terminology, and computer science definitions.

Bulk Edit

A feature allowing users to add or modify multiple wheel options simultaneously by pasting multi-line text input.

Decision Fatigue

The cognitive decline in decision-making quality following prolonged periods of decision-making, relieved by automated random choices.

DefinedTermSet

A Schema.org structured data format used to publish authoritative technical glossaries for search engines and AI answer engines.

Elimination Mode

A game mode where winning options are automatically removed from the wheel after being selected, ensuring each item wins only once.

Kleroterion

An ancient Athenian stone lottery device used in the 5th century BC to randomly assign citizens to public office and jury duty.

PRNG (Pseudorandom Number Generator)

An algorithm that uses mathematical formulas to produce sequences of numbers that simulate true statistical randomness.

Random Seed

An initial numerical value fed into a PRNG algorithm to initiate a reproducible pseudorandom sequence.

Rota Fortunae

The 'Wheel of Fortune' concept in medieval philosophy symbolizing the unpredictable rotation of fate and equal chance.

Sortition

The selection of political officials or decision outcomes by random lottery rather than voting or appointment.

Weighted Wheel

A decision wheel configuration where individual options have unequal segment sizes corresponding to customized selection probabilities.
